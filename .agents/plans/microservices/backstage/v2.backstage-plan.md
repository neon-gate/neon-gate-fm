# PURPOSE

Backstage provides real-time visibility into the ingestion pipeline.

It consumes pipeline events and streams them to developer tooling and monitoring UIs.

Initial version (V2):

```
1. Mock pipeline events
2. Broadcast via WebSocket
3. Pulse UI subscribes and renders in real-time
```

Future versions will persist pipeline state and consume NATS.

Backstage **does NOT**:

```
process audio
run AI cognition
orchestrate the pipeline
store media files
trigger ingestion commands
```

Backstage is a **pipeline event broadcaster** (read-model + monitoring).

# INITIAL MOCK STRATEGY

During early development the pipeline may not be fully running.

Backstage must include a mock event generator.

The generator emits simulated pipeline events with configurable delays.

Example flow:

```
track.upload.received
track.upload.validated
track.upload.stored
track.uploaded
track.petrified.generated
track.fort-minor.started
track.fort-minor.completed
track.approved
track.transcoding.started
track.transcoding.completed
track.ready
```

Events emitted with delays (e.g. 1.2s) to simulate real pipeline execution.

Config:

```
MOCK_MODE=true   — enable mock generator
NATS_URL unset   — also enables mock (no NATS available)
```

# USE CASE

### broadcast-pipeline-event.use-case.ts

```ts
import { EventStreamPort, PipelineEventPayload } from "../ports/event-stream.port"

export class BroadcastPipelineEventUseCase {

  constructor(
    private readonly stream: EventStreamPort
  ) {}

  async execute(input: {
    trackId: string
    eventType: string
    payload: Record<string, unknown>
  }): Promise<void> {

    const payload: PipelineEventPayload = {
      type: "pipeline.event",
      event: input.eventType,
      trackId: input.trackId,
      timestamp: new Date().toISOString(),
      payload: input.payload ?? {}
    }

    await this.stream.broadcast(payload)
  }
}
```

```ts
import { Server } from "socket.io"
import {
  EventStreamPort,
  PipelineEventPayload
} from "../../application/ports/event-stream.port"

export class SocketIOEventStreamAdapter implements EventStreamPort {

  constructor(
    private readonly io: Server
  ) {}

  async broadcast(event: PipelineEventPayload): Promise<void> {

    this.io.emit("pipeline.event", event)
  }
}
```

# MOCK EVENT GENERATOR

### mock-event-generator.service.ts

Simulates ingestion pipeline events.

```ts
import { Injectable, OnModuleInit } from "@nestjs/common"
import { BroadcastPipelineEventUseCase } from "../../application/use-cases/broadcast-pipeline-event.use-case"

const MOCK_EVENT_SEQUENCE = [
  "track.upload.received",
  "track.upload.validated",
  "track.upload.stored",
  "track.uploaded",
  "track.petrified.generated",
  "track.fort-minor.started",
  "track.fort-minor.completed",
  "track.approved",
  "track.transcoding.started",
  "track.transcoding.completed",
  "track.ready"
]

const MOCK_DELAY_MS = 1200

@Injectable()
export class MockEventGeneratorService implements OnModuleInit {

  constructor(
    private readonly broadcastUseCase: BroadcastPipelineEventUseCase
  ) {}

  async onModuleInit(): Promise<void> {

    const trackId = crypto.randomUUID()

    for (const eventType of MOCK_EVENT_SEQUENCE) {

      await new Promise((r) => setTimeout(r, MOCK_DELAY_MS))

      await this.broadcastUseCase.execute({
        trackId,
        eventType,
        payload: {}
      })

      console.log("[Backstage] Mock event:", eventType, trackId)
    }
  }
}
```

Later this will be replaced or complemented by the NATS event subscriber.

# MODULE

### backstage.module.ts

```ts
import { Module } from "@nestjs/common"
import { PipelineGateway } from "./interface/gateways/pipeline.gateway"
import { MockEventGeneratorService } from "./infrastructure/mock/mock-event-generator.service"
import { BroadcastPipelineEventUseCase } from "./application/use-cases/broadcast-pipeline-event.use-case"
import { EventStreamPort } from "./application/ports/event-stream.port"
import { SocketIOEventStreamAdapter } from "./infrastructure/websocket/socketio-event-stream.adapter"

@Module({
  providers: [
    PipelineGateway,
    BroadcastPipelineEventUseCase,
    MockEventGeneratorService,
    {
      provide: EventStreamPort,
      useFactory: (gateway: PipelineGateway) =>
        new SocketIOEventStreamAdapter(gateway.server),
      inject: [PipelineGateway]
    }
  ]
})
export class BackstageModule {}
```
