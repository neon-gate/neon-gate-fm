## Remediation

1. Add `interface LoginFormProps extends React.ComponentProps<'div'> {}` above `LoginForm` and destructure `props` on first line.
2. Audit all `useEffect` usages; replace with event-driven patterns, custom hooks that encapsulate side effects, or document exceptions.
3. Replace `ReasoningContext` and `ContextContext` with Jotai atoms; refactor consumers to use `useAtomValue`/`useSetAtom`.
4. Refactor `infra/shadcn/components/ai-elements/context.tsx` and `reasoning.tsx` to use function declarations; or document shadcn as exempt from this rule.
5. Extract inline `onClick` in `input-group.tsx:57` to a named function; or document shadcn as exempt.
6. (Optional) Consider replacing `useState` with `useImmer` in `reasoning.hooks.ts` for consistency.
